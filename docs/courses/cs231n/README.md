## Study Order For Spatial Intelligence

1. Python/Numpy and image classification
2. Linear SVM and Softmax
3. Optimization and backpropagation
4. Fully connected networks
5. CNN layers and architectures
6. Training, normalization, regularization, and transfer learning
7. Detection, segmentation, and dense prediction
8. RNNs, attention, and Transformers
9. Generative and self-supervised representation learning
10. Vision backbone to CS231A geometry and SLAM handoff

For Cyrus, treat CS231n as the visual representation backbone. CS231A explains geometry; CS231n explains the neural feature machinery that feeds modern spatial intelligence systems.
